# token-counter-VSCode

margen de error +/- 5-10% de tokens

## Instalar
1. decargar el archivo vsix

2. desde VS Code: Ctrl+Shift+P → "Extensions: Install from VSIX..."
o
```bash
code --install-extension token-counter-0.1.0.vsix
```
